Для того чтобы добавить локализацию в свой мод, нужно перейти в каталог "lang" в resources. Однако если у вас ещё нет подготовленной структуры в этой папке, то прочитайте этот небольшой гайд по созданию структуры в resources: [[Создание структуры папки resources]].
В каталоге "lang" нужно создать .json файл с названием, соответствующим коду языка. Для русского перевода это будет файл ru_ru.json, а для английского — en_us.json. Код языка должен быть написан строчными буквами, а также иметь нижнее подчёркивание вместо тире.

Теперь зайдите в созданный файл и (как и в любом .json файле) создайте фигурные скобки, если их ещё нет — всё содержимое таких файлов должно быть внутри них. Далее все последующие записи в этом файле будут представлять собой пару «ключ-значение» в формате "ключ" : "значение", отделяемые через запятую. Например:
```json
{  
  "item.myfirstmod.item1" : "Предмет 1",
  "item.myfirstmod.item2" : "Предмет 2"  
}
```
