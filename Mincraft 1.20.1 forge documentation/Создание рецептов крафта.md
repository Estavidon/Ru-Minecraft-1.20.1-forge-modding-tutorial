Для начала прочитайте [[Создание структуры папки resources|гайд]] по созданию структуры папки **resources**.
Теперь зайдите в папку resources.data.recipes и создайте там json файл с произвольным названием (но желательно похожим на название крафтового предмета, для избежания путаницы). Теперь зайдите в этот файл и создайте там примерно вот такую структуру:
```json
{  
  "type": "minecraft:crafting_shaped",  
  "category": "misc",  
  "key": {  
    "P": {  
      "item": "minecraft:diamond"  
    }  
  },  
  "pattern": [  
    "Z#Z",  
    "#P#",  
    "Z#Z"  
  ],  
  "result": {  
    "item": "minecraft:nether_star",  
    "count": 1  
  },  
  "show_notification": true  
}
```
